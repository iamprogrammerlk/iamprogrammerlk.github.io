# [Custom Currency for WooCommerce][custom-currency-for-woocommerce]

### [Custom Currency for WooCommerce][custom-currency-for-woocommerce] allows you to change the currency symbol used in [WooCommerce][woocommerce] and you can add a new custom currency type to the [WooCommerce Plugin][woocommerce-plugin].
